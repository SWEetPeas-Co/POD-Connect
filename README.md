# POD-Connect
## **by SWEetPeas Co.**

## **Objective**

Communities, clubs, and interest groups frequently struggle to keep track of their events and activities. Important information such as dates, locations, and attendance details can easily become scattered across emails, messaging platforms, or social media posts. This fragmentation makes it difficult for members to stay informed and participate in events.

The objective of this project is to develop a centralized event management platform that simplifies how organizations create, manage, and discover events. The application allows clubs to organize their activities in one accessible system while enabling users to browse events, RSVP, and stay updated.

The platform focuses on providing a clean and interactive user experience while supporting multiple user roles and a database-backed system. Rather than replicating a full commercial platform, the goal is to create a functional, scalable prototype that demonstrates how event management can be streamlined for communities and organizations.

---

## **Solution Overview**

This project addresses the challenge of disorganized event communication by creating a **club organizer application** that consolidates event information into a single platform.

The application allows organizations to:

- Create and manage clubs
- Organize events associated with those clubs
- Track event attendance and participation
- Provide a central location where users can easily discover upcoming events

By centralizing this information, the system reduces confusion and ensures that event details are consistently accessible to all members.

## **Core Features**

- User authentication and login system
- Role-based access for organizers and members
- Club creation and management
- Event creation and editing
- Event discovery and browsing
- RSVP and attendance tracking
- Centralized database for storing clubs, users, and events
- Dynamic updates when events or club information change

---

## **Technology Stack**

- **Frontend:** React Native with Expo  
- **Backend:** Firebase and MongoDB
- **Database:** Centralized database for user, club, and event data  
- **Platform:** Mobile-first design with optional desktop support
